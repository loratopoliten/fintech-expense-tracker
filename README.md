# FinTech Expense Tracker

A simple finance tracking system built in Python that allows users to record income and expenses and monitor their financial balance.

## Features
- Record income and expenses
- Categorize transactions
- View transaction history
- Calculate total balance

## Technologies
- Python
- File Handling
- CLI-based interface

## Future Improvements
- Data visualization (charts)
- Web interface (PHP/HTML/CSS)
- Database integration (MySQL)
